Mapping_Earthquakes

For this project I used JavaScript and APIs to create an interactive map that displays major earthquakes all over the world. Using Javascript, D3, and Leaflet libraries, I was able to traverse and retreive the data. The data was all plotted on a mapbox map through an API request. The result is a map that shows the magnitude and location of each earthquake, with markers that are sized and colored based on magnitude. The map also allows for users to filter the data, and select different maps, all with a legend that provides guidance to the users.
